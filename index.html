Physical Asset Verification App

User Guide

info 1. Introduction

The Physical Asset Verification App is a web-based tool designed for tracking and managing industrial assets. It allows users to:

  • Record asset details (ID, description, condition, location, etc.)
  • Upload photos of assets
  • Export/import data in Excel format
  • Create and restore backups
  • Customize form fields

This guide explains how to use all features of the app effectively.

play_arrow 2. Getting Started

2.1 Accessing the App

  • Open the app in a modern web browser (Chrome, Firefox, Edge, etc.).
  • No installation required—works offline after initial load.

2.2 Navigation

The app has three main tabs:

  • Asset Verification – Add/edit asset records
  • Data Sheet – View all assets in a table
  • Field Management – Customize form fields

check_circle 3. Asset Verification

3.1 Adding a New Asset

  • Click "Add New Asset" (or select an existing one from the dropdown).
  • Fill in the details:
    • Asset Number (required)
    • Description, Quantity, Cost Center, etc.
    • Condition (Good/Poor) – If "Poor," add notes.
    • Date Inspected (auto-filled to today).
  • Upload a photo (drag & drop or click to browse).
  • Click "Save Asset".

3.2 Editing an Asset

  • Select an asset from the dropdown.
  • Modify any details.
  • Click "Save Asset".

3.3 Resetting the Form

  • Click "Reset Form" to clear all fields.

table_chart 4. Data Sheet Management

4.1 Viewing All Assets

  • Switch to the "Data Sheet" tab to see a table of all assets.
  • Use the "View", "Edit", and "Delete" buttons for each asset.

4.2 Exporting Data to Excel

  • Click "Download Data".
  • An Excel file (Asset_Verification_YYYYMMDD_HHMM.xlsx) will download.

4.3 Importing Data from Excel

  • Click "Upload Data".
  • Select an Excel file (.xlsx or .csv).
  • The app will import and update the asset list.

4.4 Clearing All Data

    .
  • Confirm in the pop-up.
Warning: This cannot be undone!

backup 5. Backup & Restore

5.1 Creating a Backup

  • Click "Create Backup".
  • A .json file will download (Asset_Verification_Backup_YYYYMMDD_HHMM.json).

5.2 Restoring from Backup

  • Click "Restore Backup".
  • Select a previously saved .json file.
  • Confirm the restore in the pop-up.

settings 6. Field Management (Customization)

6.1 Adding a New Field

  • Go to the "Field Management" tab.
  • Enter a Field Name and select a Field Type (Text, Number, Date, etc.).
  • Click "Add Field".

6.2 Editing/Deleting Fields

  • Click the pencil (✏️) to edit a field name.
  • Click the trash (🗑️) to remove a field.

help 7. Troubleshooting

Issue Solution
Excel import fails Ensure the file matches the expected format.
Backup restore fails Check if the file is corrupted.
Photos not uploading Try a different image format (JPEG/PNG).
App not loading Refresh the page or clear browser cache.

This app provides a complete solution for asset verification with:

  • ✔ Easy data entry
  • ✔ Excel export/import
  • ✔ Backup & restore
  • ✔ Customizable fields

For support, contact your system administrator.
